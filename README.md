# UserHub

UserHub is a simple **User Management Web Application** built with **Spring Boot**.  
It supports user registration, login, profile management, and admin-controlled user listing.  
This project is ideal for beginners to practice Spring Boot, Spring Security, and JPA.

## Features

- **User Registration**: Users can sign up for an account.
- **User Authentication**: Secure login for registered users.
- **Profile Management**: Users can update their profile information.
- **User Listing**: Admins can view all registered users.
- **User Roles**: Role-based access control (admin vs regular user).

## Technology Stack

- **Spring Boot**: Application framework
- **Spring Security**: Authentication and authorization
- **Spring Data JPA**: Database access
- **MySQL**: Database
- **Thymeleaf**: Template engine for web pages
- **Bootstrap**: Frontend styling
